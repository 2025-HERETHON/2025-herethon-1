### Background

현재 우리 사회는 정치나 젠더 같이 중요한 이슈임에도 불구하고, 이를 대화에서 금기시하는 분위기가 만연합니다.</br>
그렇다면 온라인 공간은 어떨까요? 상황이 더 좋지 않습니다.</br>

### Problem & Solution
<img width="1920" height="1080" alt="4_ Problem" src="https://github.com/user-attachments/assets/8befc394-bb1a-4a0a-b2fa-0dfa24cd36ab" />
</br>
</br>
우리는 왜 이러한 주제로 대화하기 어려웠을까요?
이를 알기 위해 FGI를 진행했고, 세 가지 페인포인트를 도출했습니다.

첫번째로 관심은 있지만, 정보 탐색과 이해에 투자할 여력이 없다는 점
두번째로 관련 이슈로 대화할 때, 소통이 단절된 경험이 많았다는 점
세번째로 본인의 입장은 있지만, 이를 설득력 있게 표현하기 어렵다는 점 입니다.

<img width="1920" height="1080" alt="5_Solution" src="https://github.com/user-attachments/assets/9612415c-950a-45f2-9f30-ffd9f3b310c9" />

저희 서비스 ‘사상사이’는 이 세 가지를 ① 아티클 큐레이션, ② 토론형 커뮤니티, ③ AI 리서치 도구를 통해 해결합니다.

### 1. 아티클 큐레이션
<img width="1920" height="1080" alt="10 GUI" src="https://github.com/user-attachments/assets/733d6090-478a-4c9f-95bb-06ce01fb2379" />
<img width="1920" height="1080" alt="11 GUI" src="https://github.com/user-attachments/assets/5ab9d345-3bd9-4304-9625-b8dd2c548b28" />


아티클 큐레이션에서는 전문가가 특정 이슈에 대해 작성한 아티클을 주제별로 제공하며,
사용자는 한 줄 요약을 통해 빠르게 핵심을 파악할 수 있습니다.
또한 우측에는 해당 아티클에서 파생된 논제에 대하여 토론하는 글도 함께 확인할 수 있습니다.

### 2. 토론형 커뮤니티
<img width="1920" height="1080" alt="12 GUI" src="https://github.com/user-attachments/assets/056cd2de-9243-4c60-bdc5-30ee741afe07" />
<img width="1920" height="1080" alt="13 GUI" src="https://github.com/user-attachments/assets/2eac85ba-a397-4ad5-9fbf-e948770f4e8e" />

일방적 주장만 오가며 소통이 단절되던 경험을, 상대의 의견을 경청해야 하는 토론 형식을 통해 해소하고자 했는데요.
사용자는 주어진 논제에 대해 찬/반/중립 중 하나를 선택해 투표하고,
하단의 ‘모아보기’ 영역에서 각 입장의 대표 주장을 한눈에 파악할 수 있습니다.
또한 ‘더보기’ 버튼을 누르면 각 입장의 댓글을 필터링하여 확인할 수 있습니다.

### 3. AI 리서치 도구
<img width="1920" height="1080" alt="15 GUI" src="https://github.com/user-attachments/assets/fd4e5280-cc67-46da-9e75-cc817ff44d0a" />
주장을 작성한 뒤 AI 버튼을 누르면, 이를 뒷받침할 수 있는 논문, 행정 자료 등을 근거로 제공받습니다.
이를 통해  ‘학습 → 토론 → 정책 제안’의 과정을 AI 리서치 도구와 함께 하나의 흐름 속에서 깊게 경험할 수 있게됩니다.
이러한 연계형 시스템을 통해 기존 커뮤니티가 제공하지 못했던 성숙한 사회 담론의 장을 제공합니다.

### 개발환경에서의 실행 방법
  ```
  .env 파일 생성 후 .env.example 파일 참고해서 실제 값으로 변경
  $ cd theCommunityProject/
  $ pip install -r requirements.txt
  (버전이 맞지 않을 경우 $ pip install django 명령어 실행 후 위 명령어 실행)
  $ python manage.py makemigrations
  $ python manage.py migrate
  $ python manage.py runserver
  ```
